fromJson

fromJson(json, [options]) : [DomainObject](class-DomainObject.html)

Наполняет текущий объект данными в виде json. По умолчанию обновляются значения оригинальных свойств (управляется опцией dirty).

Parameters:

Name Type Default Attributes Description
json Object Данные объекта в виде json
options Object optional Дополнительные опции
options.dirty Boolean optional Признак Обновляются текущие свойства объекта; иначе - оригинальные
options.norollback Boolean true optional Значения не копируются в коллекцию оригинальных значений; имеет смысл, только если задана опция dirty
options.partial Boolean optional Означает, что json-данные содержат не все необходимые свойства.

По умолчанию, если не задана опция dirty, то объект становится загруженным (устанавливается флаг isLoaded). Кроме того, если ts в json-данных больше текущего, то также удаляются все свойства, которых нет в json. Оба этих действия отключаются опцией partial. |

Returns:

Текущий объект Type: DomainObject

subscribe(eventName, callback, [context]) : Object

Подписывает обработчик callback на событие eventName и возвращает disposable объект, который отписывает этот обработчик в методе dispose. Метод аналогичен методу bind, но возвращает disposable объект, который выполняет unbind.

Parameters:

Name Type Default Attributes Description

Returns:

unbind (inherited from Observable)

unbind(eventName, [callback], [context]) : *

Отписывает обработчик callback от события eventName. Обработчик отписывается, только если он был подписан с тем же значением параметра context. Если задано только eventName, то от события eventName отписываются все подписанные обработчики. Если не заданы eventName, callback и context, то все подписанные обработчики отписываются от всех событий. Если заданы eventName и context, то от события eventName отписываются все обработчики контекста. Если задан только context, то отписываются все обработчики контекста от всех событий.

Parameters:

Name Type Default Attributes Description
eventName String
callback function optional
context Object optional

Returns:

Type: *
