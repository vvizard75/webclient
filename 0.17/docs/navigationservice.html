NavigationService

NavigationService - это компонент, который используется партами для перехода и возврата в/из вложенных контекстов. Например, при открытии вложенных редакторов. Экземпляр NavigationService парту устанавливает регион, при активации парта. В регион экземпляр NavigationService устанавливает RegionManager при добавлении в него нового региона. Для вложенного региона (региона, являющегося партом), по умолчанию используется NavigationService из родительского региона, но может быть заменена на собственную.

Для перехода во вложенный контекст парт вызывает метод navigate. Пример открытия вложенного редактора для создания нового объекта из редактора навигируемого свойства:

that.navigationService.navigate({
                part: args.partName || "ObjectEditor:" + valueObj.meta.name,
                partOptions: {
                    viewModel: valueObj,
                    editorContext: that._createNestedEditorContext()
                },
                onReturn: function(result) {
                    // on come back refresh the list
                    if (result.success) {
                        that.raiseItemsChanged({added: [result.object]})
                    } else {
                        that.value().remove(valueObj);
                    }
                }
            });

В метод navigate передается имя или экземпляр парта, который нужно активировать, параметры для него и опциональный колбэк, который будет вызван при возврате. При вызове navigate текущий парт суспендится. Для этого он выгружается, но не удаляется из региона. При выгрузке не происходит вызова метода queryUnload. Метод unload вызывается с параметром \{reason: "suspend"\}. В независимости от результата unload текущий парт не выгружается. Открывшийся вложенный парт для возврата "назад" вызывает метод close у NavigationService. При этом происходит выгрузка текущего (вложенного) парта, также без вызова queryUnload. После этого восстанавливается предыдущий парт (для него вызываются методы activating/activated с параметром \{reason:"resume"\}. После завершения восстановления парта вызывается колбэк, который он передал ранее в метод navigate с результатом, переданным вложенным партом при вызове close, в качестве параметра.

Навигация с помощью NavigationService связана с глобальной навигацией (см. Navigation). При суспенде парта (когда он вызвал navigationService.navigate), берется текущее состояние приложения и сохраняется. Позднее при возврате (вложенный парт вызвал navigationService.close), выполяется замена глобального состояния приложения на сохраненное. Таким образом, история навигации браузера, при возврате из вложенного контекта очищается. Пример перехода по цепочке вложенных состояний: s1 (история пустая) =navigate=> s2 ("s1") =navigate=> s3 ("s2", "s1") =close=> s2 ("s1") =close=> s1 (история пустая)

API

navigation: function(options) {}

/**
 * Unload current part and return to previous suspended part.
 * @param result {Object} Any object to pass into callback in onReturn options' field passed into navigate method earlier.
 */
close: function(result) {}
