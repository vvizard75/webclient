peBoolean

Для свойства типа boolean существует несколько редакторов:

  • peBoolean - в виде чекбокса (checkbox)
  • peBooleanSwitch - в виде переключателя (switch)

Выбор редактора осуществяется с помощью опции presentation:

  • checkbox - peBoolean
  • switch - peBooleanSwitch

Если опция presentation не задана, то выбирается редактор по умолчанию. Каждая из реализаций устанавливает себя как редактор по умолчанию свойства типа boolean. Поэтому при незаданном presentation будет выбран тот редактор, который подключен позднее (если подключены оба). Чтобы во всем приложении всегда использовать одну из реализаций досточно оставить подключение соответствующего js файла (peBoolean.js или peBooleanSwitch.js) в свой main.js. Если в приложении требуются обе реализации, то реализацию по умолчанию можно также задать в статическом объекте PropertyEditor.DefaultMapping:

PropertyEditor.DefaultMapping["boolean"] = peBoolean;

Оба редактора поддерживают две опции, влияющие на поведение, nullable и threeStates (добавлена в версии 0.16). В зависимости от сочетания опцию, редакторы позволяют либо задать значения true/false, либо true/false/null, либо true/null:

nullable threeStates Режим
false undefined threeStates := false
true undefined threeStates := true
false true недопустимо (как сохранить 3-е состояние в notnull свойстве?)
true true PE с 3-мя состояниями true/false/null
false false PE с 2-мя состояниями true/false
true false PE с 2-мя состояниями true/null

peBoolean

Редактор свойств типа boolean в виде чекбокса (input type="checkbox"). Включается опцией presentation="checkbox" в метаданных свойства. Поддерживает nullable свойства, т.е. если свойство nullable=true и устанавливаемое значение null - редактор переходит в неопределенное (indeterminate) состояние. Так же, редактор nullable свойства при клике по нему циклически меняет состояние (и принимает соответствующие значения): checked (true), unchecked (false), indeterminate (null).

Для реализации неопределенного состояния в <input type='checkbox'/> используется html5 атрибут indeterminate. К сожалению этот атрибут поддерживается не всеми браузерами: читать раз, читать два. Android stock broswer (tested in Android 2.0 and 4.0) doesn't support the property at all, and Safari mobile on iOS 3.1 doesn't have support for it as well

peBooleanSwitch

Редактор свойства типа boolean в виде переключателя. Представляет собой переключатель с двумя или тремя состояниями. Третье состояние добавляется, если свойство является nullable.
