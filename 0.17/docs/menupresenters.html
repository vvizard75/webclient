MenuPresenters

Для отображения Menu используются презентеры:

  • MenuPresenter
  • MenuNavPresenter
  • MenuButtonsPresenter
  • DropDownMenuPresenter
  • ButtonDropDownPresenter

Важным моментом является то, что презентеры (MenuPresenter, MenuNavPresenter, MenuButtonsPresenter) используют общий базовый класс MenuPresenterBase.

Опции

Для презентеров MenuPresenter, MenuNavPresenter, MenuButtonsPresenter поддерживаются общие опции:

Опция Тип Значение
cssClass String имя css-класса для корневого dom-элемента
itemCssClass Strung имя css-класса для dom-элемента элемента меню
reverse Boolean Если true, то пункты будут выводится в обратном порядке
itemPresentation "icon", "text", "both" Представление элемента меню: только иконка, только текст, иконка и текст
tooltips Boolean Выводить тултипы для элементов. По умолчанию true
radio Boolean Режим radio - то же, что задается для Menu. По умолчанию false.
noFocus Boolean true чтобы задать tabIndex=-1 для всех элементов. По умолчанию false.

MenuPresenter

Большинство компонентов, содержащих меню, используют MenuPresenter. MenuPresenter отображает меню в "usethics" стиле: Стили меню находятся в menu.less/css и не используют стили Bootstrap. Меню существует в двух вариантах:

  • "обычное" - белый фон, черные буковки (верхнее меню на скриншоте)
  • "контрастное" - зелый фон, белые буковки (нижнее меню на скриншоте)

"Контрастный" вариант получается добавлением стиля x-menu--contrast к контейнеру меню. Пример:

<div class='x-editor-menu-container x-menu-bar x-menu--contrast'>
{{view class="ui.MenuPresenter" viewModel=this.menu tag="div" cssClass="x-editor-menu"}}
</div>

MenuPresenter предназначен для меню операций. Он использует элементы
