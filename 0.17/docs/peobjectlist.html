peObjectList

Использование

peObjectList - редактор массивного навигируемого свойства. Используется по умолчанию для всех свойств с условием: vt == "object" && many.

Параметры

См. также настрйоки, общие для Редакторов навигируемых свойств

Наименование Значение Обязательность, значение по умолчанию
columns Массив описаний колонок (см. ниже). Описание колонки аналогично ObjectList Необязательно. Если не задано, создаются колонки для каждого примитивного свойства объекта
menuList Метаданные меню списка Необязательно. Если не задано, то используется меню из статического поля core.ui.Menu.Defaults.peObjectListProperty
menuRow Метаданные меню строки Необязательно. Если не задано, то используется меню из статического поля core.ui.Menu.Defaults.peObjectListRow
orderBy массив наименований колонок, по которым выполняется сортировка, аналогично ObjectList Необязательно
where Функция для клиентской фильтрации. Принимает объект из значения свойства, возвращает Boolean (true - объект отображается в свойстве, false - объект скрывается из свойства) Необязательно
indexed true/false - признак индексированного свойства Необязательно, по умолчанию false
layout json-объект, см. Property Editors Необязательно. По умолчанию

Селектор

При создании ObjectSelector'а при вызове команды Select редактора свойства, в качестве имени парта используются:

  1. свойство partName аргументов команды Select (если передано);
  2. свойство objectSelector.name параметров редактора (если задано);
  3. строка - "ObjectSelector:" + имя типа значения.
