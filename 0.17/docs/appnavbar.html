AppNavBar

Типовым случаем является наличие в приложении тулбара вверху окна. Данный тулбар не является неким одним компонентом, а является лишь разметкой, внутри которой создаются другие компоненты. Типом случаем является:

  • SystemMenu - меню, в которое модули добавляют общесистемные сервисы, в частности Черновики и главное меню навигации
  • AppNavToolbar - см. App Navigation
  • LanguageMenu - меню переключения языка интерфейса
  • AuthMenu - меню модуля security
  • OnlineBeacon - меню модуля Offline - см. OnlineBeacon

Создание перечисленных компонентов обычно производится при инициализации экземпляра Applicaiton - см. Application class.

Однако, чтобы все компоненты выглядели в тулбаре в едином стиле, введены css-классы, которые подразумевают некоторую типовую структуру html-разметки. Основным классом является x-app-navbar, он определяет основной "look-n-feel":

Дополнительно к этому классу могут задаваться классы-модификаторы, изменяющие тему тулбара:

  • x-app-navbar--dark - темная тема:
  • x-app-navbar--light - светлая тема на основе палитры Bootstrap:

Типовая html-разметка (из шаблона _Layout.cshtml):

<div class="root">
    <div class="x-app-navbar" role="navigation">
        <div class="navbar-header">
            <a class="navbar-brand" href="~/">@Model.Config.AppName</a>
            <button type="button" class="navbar-toggle collapsed" data-toggle="collapse" data-target="#app-navbar-collapse">
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
                <span class="icon-bar"></span>
            </button>
        </div>

        <div id="app-navbar-collapse" class="container collapse navbar-collapse">
            <ul class="x-sys-menu nav navbar-nav" id="sysMenu"></ul>
            <ul class="nav navbar-nav navbar-left visible-lg visible-xs" id="areasMenu"></ul>
            <ul class="nav navbar-nav navbar-right">
                <li id="lang-menu" class="x-lang-menu"></li>
                <li id="account-menu"></li>
                <li id="online-beacon"></li>
            </ul>
        </div>
    </div>
...
</div>

Здесь элементы для всех основных компонентов тулбара, перечисленные выше, убраны в группирующий элемент с id app-navbar-collapse. Это делается для поддержки Response Design - при уменьшении размеров окна, все элементы тулбара будут скрыты, а вместо них появится кнопка "navbar-toggle":
