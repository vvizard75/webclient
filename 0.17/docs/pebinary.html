peBinary

Редактор свойства типа binary.

Для свойств с contentType="image" (см. Параметры) peBinary отображает thumbnail изображения, при клике на который, открывается preview. Preview реализован с помощью сторонней библиотеки jQuery ColorBox.

Параметры
Наименование Значение Значение по умолчанию Комментарий
contentType cтрока undefined ContentType хранящихся данных. Определить можно (как и любой фасет) сразу для свойства (model-ext.js)

metadata.entities.User.props.avatar.contentType = "image"; | | width | число | undefined | ширина изображения, получаемого с сервера для preview | | height | число | undefined |высота изображения, получаемого с сервера для preview | | thumbnailWidth | число | 256 | максимальная ширина thumbnail (атрибут max-height элемента <img />) | | thumbnailHeight | число | undefined | максимальная высота thumbnail (атрибут max-height элемента <img />) | | menu | опции меню (см Menu) | undefined | для расширения стандартного меню pe | | commands | dictionary экземпляров команд | undefined | для расширения стандартных команд pe | | imgPreviewSettings | object | | опции colorBox | | imgPreviewSettings.imgError | строка | resources["peBinary.previewLoadingError"]("peBinary.previewLoadingError".html) ("Ошибка загрузки изображения"/"Image loading error") | сообщение об ошибке при открытии preview |

На текущий момент параметры width, height, thumbnailWidth и thumbnailHeight не поддерживаются сервером. Т.е. на клиент изображение приходит всегда в оригинальном размере
